# Видеоплеер на JS

Прототип видеоплеера

## Запуск livereload для разработки

* Установка зависимостей

```bash
>>> pip install -r requirements.txt
```

* Запуск liverload

```bash
>>> livereload src
```

* [127.0.0.1:35729](http://127.0.0.1:35729/)
