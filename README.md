# Видеоплеер на JS

Прототип видеоплеера

## Запуск livereload для разработки

* Установка зависимостей

`pip install -r requirements.txt`

* Запуск liverload

`livereload src`

* [127.0.0.1:35729](http://127.0.0.1:35729/)
